# 🗣️ Elkarrizketa 3 (Irakaslea 3 - Lehen Hezkuntza)

Nik metodologia tradizionalagoa erabiltzen dut, baina aldaketak sartzen saiatzen ari naiz. Talde-lana probatu dut eta ikusi dut ikasle batzuek gehiago parte hartzen dutela. Hala ere, beste batzuek nahiago dute bakarka lan egin. Uste dut ikasleen profilaren araberakoa dela.
