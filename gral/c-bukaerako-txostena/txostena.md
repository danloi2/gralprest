## Hasierako Atalak

:::{glossary}
Portada
: Txostenaren identifikazio-orria (izenburua, egilea, tutorea...).

Laburpena
: Ikerketaren sintesia hiru hizkuntzetan (euskaraz, gazteleraz eta ingelesez).

Eskerrak
: Ikerketaren lagun duten pertsonei eskainitako atala.

Aurkibideak
: Edukiak, taulak eta irudiak non dauden adierazteko zerrendak.
:::

Txostenaren hasieran ondoko lau elementu doaz, orrialdeetan zenbakirik gabe:

::::{grid} 1 1 2 2

:::{card} 📋 Portada
- Unibertsitatearen logoa
- Gradua eta ikasturtea
- GRALaren izenburua (euskaraz eta gazteleraz)
- Egilearen izen-abizenak
- Tutorearen izen-abizenak
- Data
:::

:::{card} 📝 Laburpenak eta Hitz Gakoak
Hiru hizkuntzetan (150-250 hitz bakoitzak):
- Euskaraz (*Laburpena*)
- Gazteleraz (*Resumen*)
- Ingelesez (*Abstract*)

+ **Hitz Gakoak / Keywords** (5-7 hitz)
:::

:::{card} 🙏 Eskerrak
Aukerakoa. Eskerrak ematea:
- Tutoreari
- Ikertutako pertsonei
- Laguntzaile akademikoei
:::

:::{card} 📑 Aurkibideak
- Eduki-aurkibidea
- Taula-aurkibidea
- Irudi-aurkibidea
- Laburdurak (baldin badaude)
:::
