:::{glossary}
Eranskinak
: Testu nagusian sartzen ez den baina osagarri gisan beharrezkoa den materiala (baimenak, tresnak, transkripzioak…).
:::

Eranskinak **aukerakoak** dira. Baldin badaude, zenbakituta eta testuan aipaturik egon behar dira.

---

## Eranskinetan Sartu Daitekeena

| Eranskina | Edukia |
|-----------|--------|
| **Eranskina A** | Baimenen formularioak |
| **Eranskina B** | Galdetegiak / Elkarrizketa gidoiak |
| **Eranskina C** | Transkripzioen laburpenak |
| **Eranskina D** | Materialen adibideak |
| **Eranskina E** | Software irteeren pantaila-argazkiak |

---

## Eranskinak Formateatzea

- Eranskina bakoitza **orri berri batean** hasi
- Testuan aipatu: *(ikus Eranskina A)* edo *(ikus Eranskina B)*  
- Alfabetikoki zenbakitu (A, B, C… edo Arabiar zenbakiak: 1, 2, 3…)
