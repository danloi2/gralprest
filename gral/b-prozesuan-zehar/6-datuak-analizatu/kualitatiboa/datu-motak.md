Analisi kualitatiboan, ez dugu zenbakiak aztertzen, baizik eta **esanahia duten datuak**: pertsonen hitzak, testuak, irudiak edo ekintzak.

```{admonition} Ideia nagusia
:class: important
Edozein material bihur daiteke datu kualitatibo, baldin eta interpretatzen badugu.
```

---

## Datu kualitatibo motak

Hezkuntzan eta ikerketan, hainbat datu-iturri erabil daitezke:

:::{glossary}

Elkarrizketa
: Pertsona bati egindako galderen bidez iritziak, esperientziak eta sinesmenak biltzeko tresna. Formatu egituratua, erdi-egituratua edo irekia izan daiteke. Helburua da parte-hartzaileen perspektiba zuzenean jasotzea.

Behaketa
: Ikasgelan edo testuinguru naturalean gertatzen dena zuzenean aztertzea, esku hartu gabe. Helburua portaerak, interakzioak eta testuinguru dinamika ulertzea da. Teknikak behaketa sistematikoa, anotazioak edo grabaketak barne hartzen ditu {cite:p}`stake1998`.

Gelako egunkaria
: Irakasleak edo ikerlariak idatzitako dokumentu pertsonala, non esperientziak, behaketak, zailtasunak eta hausnarketak jasotzen diren. Erabilgarria da ikasgelako prozesuak eta ikaskuntza dinamika ulertzeko.

Dokumentua
: Analisirako erabilgarria den edozein testu formal: programazio didaktikoak, ikas-planak, proiektu-plangintza edo testuliburuak. Normalean proposamenak, erabakiak edo behaketa objektiboak jasotzen ditu.

Eztabaida / Focus group
: Parte-hartzaileen taldean burututako saioa, non ideiak eta esperientziak elkarren artean konpartitzen diren. Helburua informazio aberatsa lortzea interakzioak eta elkarrizketa dinamikoak erabiliz.

Ikasleen ekoizpenak
: Ikasleen koadernoak, ariketak, proiektuak edo bestelako lanak. Helburua da ikaskuntza-prozesua eta gaitasunak aztertzea, bai eta irakasleen jarraipena laguntzea.

Irudiak / material bisuala
: Argazkiak, marrazkiak, infografiak edo bestelako material bisuala. Helburua testuinguruak eta ikasketa-prozesuak modu bisualean dokumentatzea.

Audioak eta bideoak
: Grabaketak, aurkezpenak edo bestelako edukien erregistroak. Helburua elkarrizketa, interakzio eta ekintza naturalak jasotzea eta gero aztertzea.

:::

| Datu-iturria | Zer jasotzen du? |
|:---|:---|
| 🗣️ **Elkarrizketak** | Iritziak eta esperientziak |
| 👀 **Behaketak** | Benetako portaera testuinguruan |
| 💬 **Eztabaidak (focus group)** | Ideiak taldean eraikita |
| 📓 **Egunkariak** | Hausnarketak eta bizipenak |
| 📄 **Dokumentuak** | Proposamenak eta erabakiak |
| 🧑‍🎓 **Ikasleen ekoizpenak** | Koadernoak, ariketak, proiektuak |
| 🖼️ **Irudiak / material bisuala** | Argazkiak, marrazkiak |
| 🎧 **Audioak eta bideoak** | Grabaketak, aurkezpenak |

## Gure datasetan erabiliko ditugunak

Ikastaro honetan hiru datu-iturri motatako materialak aztertuko ditugu:

```{admonition} Datu mota guztiak modu berean aztertzen dira
:class: tip
Mota desberdina izanda ere, elkarrizketak, egunkariak eta dokumentuak analisi-prozesu **berari** jarraitzen diote:

- Elkarrizketak → iritziak
- Egunkariak → esperientziak
- Dokumentuak → proposamenak
```
