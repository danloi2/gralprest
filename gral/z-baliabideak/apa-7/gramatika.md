# 4. Gramatika (Grammar)

## 4.1. Aditz-denborak

- **Iragana** → egindako ikerketak
- **Oraina** → teoriak eta egia orokorrak

:::{table} APA 7 estiloan gomendatutako aditz-denborak (atalen arabera)
:label: tab:apaverb

| Lanaren atala                                    | Gomendatutako denbora | Adibidea                                         |
| ------------------------------------------------ | --------------------- | ------------------------------------------------ |
| Literaturaren berrikuspena (beste egileen lanak) | Iragana               | García (2020) azaldu zuen…                       |
|                                                  | Orainaldi perfektua   | Ikertzaile askok aztertu dute…                   |
| Metodoa (prozedura deskribapena)                 | Iragana               | Parte-hartzaileek galdetegia bete zuten          |
|                                                  | Orainaldi perfektua   | Beste ikerketek antzeko metodoak erabili dituzte |
| Emaitzen aurkezpena                              | Iragana               | Emaitzek erakutsi zuten…                         |
|                                                  |                       | Puntuazioak handitu ziren                        |
| Emaitzen interpretazioa (eztabaida)              | Oraina                | Emaitzek adierazten dute…                        |
| Ondorioak eta inplikazioak                       | Oraina                | Ondoriozta dezakegu…                             |
| Etorkizuneko ikerketak                           | Etorkizuna            | Etorkizuneko ikerketek aztertu beharko dute…     |
:::

Oharra. APA 7 estiloak gomendatzen du aditz-denborak modu koherentean erabiltzea eta testuan aldaketa arbitrarioak saihestea ([123dok.com][2]).
