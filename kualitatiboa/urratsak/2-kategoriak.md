:::{glossary}

Kategoria
: Antzeko esanahia edo gaia duten kodeen **multzokatzea**. Kodifikaziotik haratago, ideiak egituratzen laguntzen du.

Gaia
: Kategoriak analizatu ondoren azaleratzen den ideia edo eredu nagusia. Ikerketaren emaitzen ardatz egituratzaile gisa funtzionatzen du.

Dimentsioa
: Kontzeptu edo kategoria baten barruan bereiz daitekeen azpi-atal edo ikuspegi espezifikoa (adibidez, gaitasun baten alderdi ezberdinak).

:::

## Kodeetatik kategoriak eta gaiak sortzen

Behin kode asko dituzunean, hurrengo pausoa horiek antolatzea da. 

Pausoak:
Kodeak → Kategoriak → Gai nagusiak

Adibidez, agian hiru irakasle desberdinek gauza bera esan dute modu desberdinean eta hiru kode sortu dituzu. Orain horiek bildu behar dituzu.

**Adibidea:**

| Urratsea | Elementuak |
| :--- | :--- |
| **Kodeak:** | - motivación<br>- participación<br>- interacción |
| **Kategoria:** | Metodologías activas |
| **Gai Nagusia:** | Participación del alumnado |

> [!NOTE] 
> Kategoriak eta gaiak zure TFG-ko Emaitzak atalaren egitura edo azpi-atalak izango dira oso seguruenik.
