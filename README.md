# GRALprest 🎓

> **Zure gradu amaierako lana bideratzeko gida didaktikoa.**

**GRALprest** is a modern, high-performance guide designed for university students to navigate the research process of their final degree project (GRAL). It combines methodological precision with a premium academic aesthetic, providing all the necessary tools for both quantitative and qualitative analysis.

## 🏛️ Project Philosophy

The project is built on the pillars of **methodological accuracy**, **academic excellence**, and **open-access resources**. Inspired by the rigor of scientific research and the clarity of educational design, **GRALprest** offers a bridge for students to master the complexities of academic inquiry.

---

## ✨ Key Features

### 📊 Advanced Analysis Tools

- **Quantitative Analysis**: Step-by-step guides for **JAMovi** and **SPSS**, from descriptive statistics to inferential basics.
- **Qualitative Analysis**: Immersive workflow using **QualCoder**, covering coding, grouping, and interpretation.
- **Data-Driven**: Real datasets and examples provided for hands-on learning.

### 📑 Step-by-Step Guide

- **10 Essential Steps**: A comprehensive dekalogo from setting the conceptual framework to final writing.
- **Thematic Research**: Guidance on modern topics like **Artificial Intelligence** and **Universal Design for Learning (UDL)**.
- **Structured Content**: Clear categorization of phases: Before, During, and Final Report.

### 📝 Writing & Resources

- **APA 7 Support**: Specialized sections for citations, references, and academic style.
- **Ready-to-Use Templates**: Downloadable resources and templates to kickstart your thesis.
- **Searchable Glossary**: Integrated glossary for quick technical reference.

## 🚀 Getting Started

### Prerequisites

- Node.js (v24+)
- pnpm (v10+)
- Python (v3.14+)

### Installation & Build

1. **Install dependencies**:
   ```bash
   pnpm install
   ```
2. **Build the site**:
   ```bash
   pnpm run build
   ```
3. **Clean build files**:
   ```bash
   pnpm run clean
   ```
